Creating a storehouse

Storehouses are created in Configuration > Storehouse Management > Storehouse parameters.

Make sure no storehouse is selected. If you have one selected, reload the page.

All fields are mandatory. 

You identity can be different from your company or from other storehouses. This is useful if you have different divisions managing different sets of products, for instance.

If your address does not have a state, enter the name of a region.

Once the storehouse is created, you can add other users as storehouse managers.
