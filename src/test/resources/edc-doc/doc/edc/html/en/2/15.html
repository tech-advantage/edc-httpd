Creating/copying a product

Creating a product from scratch

To create a new product, go to the Product Parameters panel.

  1. Switch Active on or off to decide whether the product you are creating will be active.
    Inactive products cannot be edited.
  2. Enter a product name.
  3. Enter a version name.

    Version names need the following format: number.number.number (1.0.0, 2.8.1, etc.).
  4. Enter a documentation title.
    The title appears in the exported documentation.
  5. Enter a vendor name.
    This is useful if you have several vendors in the same storehouse.
  6. Select a product type: Eclipse RCP or Web.
    This impacts the imported documentation.

Click Clear to remove the contents of all fields. Click Create a product to create the new product.

The created product appears in the Products of EDC panel.

Creating a product from another product

To copy a product, go to the Products of EDC panel.You can use this option to create a completely new product, or just to create a new version.

 

  1. Select the product you want to copy.
  2. Click Copy. 
  3. Enter a product name and version name for the new product.

The new product appears in the list.
