Removing a storehouse manager

Storehouse managers are removed in Configuration > Storehouse Management > Storehouse Managers.

Before removing a storehouse manager, select a storehouse in the list of Storehouses.

In the list of Storehouse Managers:

  1. Select the checkbox in front of the manager you want to remove.
  2. Click DISSOCIATE to confirm.

One checkbox is grayed out. This is the reference storehouse manager defined when the storehouse was created. This manager cannot be removed.
