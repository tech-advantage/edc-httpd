Creating a new product version

To copy a product, go to the Products of EDC panel. You can use this option to create a completely new product, or just to create a new version.

 

  1. Select the product in which you want to add a version.
  2. Click Copy. 
  3. Enter the name of the product.
  4. Enter the new version name.
    Version names need the following format: number.number.number (1.0.0, 2.8.1, etc.).

The new product version appears in the list.
