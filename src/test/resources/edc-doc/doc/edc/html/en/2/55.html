Adding storehouse managers

Storehouse managers are added in Configuration > Storehouse Management > Storehouse Managers.

Before adding a storehouse manager, select a storehouse in the list of Storehouses.

In the list of Storehouse Managers:

  1. Use the Find a user field to look for a user.
  2. Select the user from the list that appears.
    If the list is too long, you can type in part of the name to filter.
  3. Click ADD AS MANAGER to confirm.
